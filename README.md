# DooR DasH: Scare vs Laugh Touchdown
A competitive 2-player board game set in the Monsters Inc. universe where Scarers and Laughers race across the Floor collecting energy from children's doors.

# Why it's interesting
Combines Snakes & Ladders-style movement with RPG elements — each monster has unique abilities, energy management, and card-based chaos that makes every game different.

# Built With
- Java
- Eclipse IDE

# How to Run
1. Clone the repo and open it in Eclipse
2. Run `Main.java`
3. Follow the prompts to choose your role and start playing
